# To-Do List ✅

A simple and interactive **To-Do List App** built using **HTML, CSS, and JavaScript**.  
This project helps users manage their daily tasks efficiently.

---

## 🚀 Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Clean and minimal UI  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
